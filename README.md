# 个人简历网站

欢迎来到我的个人简历网站！这是一个使用 Nuxt 3 构建的极简启动项目。

## 关于本项目

这个项目是一个简洁的个人简历网站，使用 Nuxt 3 作为框架，旨在展示我的专业技能和工作经历。它不仅作为一个在线简历，同时也是我技术能力的展示。

## 快速开始

### 安装依赖

请确保你已经安装了 Node.js。之后，你可以选择以下任一包管理器来安装项目的依赖：

```bash
# 使用 npm
npm install

# 使用 pnpm
pnpm install

# 使用 yarn
yarn install

# 使用 bun
bun install

启动开发服务器，你将能够在 http://localhost:3333 上查看你的个人简历网站：

# 使用 npm
npm run dev

# 使用 pnpm
pnpm run dev

# 使用 yarn
yarn dev

# 使用 bun
bun run dev
生产环境
为生产环境构建应用程序：

# 使用 npm
npm run build

# 使用 pnpm
pnpm run build

# 使用 yarn
yarn build

# 使用 bun
bun run build
本地预览生产构建
在本地预览生产构建：

# 使用 npm
npm run preview

# 使用 pnpm
pnpm run preview

# 使用 yarn
yarn preview

# 使用 bun
bun run preview
